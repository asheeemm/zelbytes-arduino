# Day 11 Multi-Sensor Dashboard

## CSV Columns

| Column | Meaning | Unit |
|----------|----------|------|
| time_ms | Milliseconds since startup | ms |
| light_raw | LDR ADC value | 0-1023 |
| temp | Temperature | °C |
| hum | Humidity | % |
| distance_cm | Ultrasonic distance | cm |
| soil_pct | Soil moisture percentage | % |
